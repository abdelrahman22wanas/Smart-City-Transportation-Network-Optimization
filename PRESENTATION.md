# Smart City Transportation Network Optimization

## Slide 1 — Title
**Smart City Transportation Network Optimization**  
CSE112 · Design and Analysis of Algorithms  
Alamein International University

---

## Slide 2 — Problem
- Cairo transportation is dense and complex
- Roads, facilities, and demand must be optimized
- The project treats the city as a weighted graph
- Goal: improve routing, transit, and infrastructure planning

---

## Slide 3 — Tech Stack
- **Backend:** Python, FastAPI, Pydantic, Uvicorn
- **Frontend:** React, Vite, Axios, CSS
- **ML:** scikit-learn, NumPy
- **Deployment:** Docker, Vercel-ready frontend

---

## Slide 4 — Data Model
- Neighborhoods
- Facilities
- Existing roads
- Potential new roads
- Traffic patterns
- Metro lines
- Bus routes and demand

---

## Slide 5 — How It Works
1. User selects a tab
2. Frontend sends an API request
3. Backend runs the algorithm
4. JSON result returns
5. UI updates with maps, tables, or summaries

---

## Slide 6 — Algorithms
- **Kruskal / Prim**: MST design
- **Dijkstra / A\***: route planning
- **Time-dependent Dijkstra**: traffic-aware routing
- **Dynamic Programming**: bus scheduling and road maintenance
- **Greedy Methods**: signals and emergency routing
- **ML Prediction**: traffic forecasting
